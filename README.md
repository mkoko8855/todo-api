devtools -> Spring boot에서 제공하는 개발 편의를 위한 모듈.

크게 5가지 기능 제공.
1. Property Defaults
2. Automatic Restart
3. Live Reload
4. Global Settings
5. Remote Applications
->
브라우저로 전송되는 내용들에 대한 코드가 변경되면, 자동으로 어플리케이션을 재시작하여 
브라우저에도 업데이트를 해주는 역할.


# spring boot devtools 설정

1. 파일 -> 설정
2. 빌드, 실행, 배포 탭 클릭 -> 컴파일러 탭 클릭
3. 자동으로 프로젝트 빌드 클릭 후
4. 왼쪽 탭에 고급 설정 클릭 후
5. 컴파일러 탭 -> 프로젝트가 실행중인 ~~auto-make 클릭
